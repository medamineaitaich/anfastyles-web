# AnfaStyles Web

Frontend-only repo extracted from the original monorepo (`apps/web`).

## Commands

- Install: `npm ci` (or `npm install`)
- Dev: `npm run dev`
- Build: `npm run build` (outputs `dist/`)
- Start (preview): `npm run start` (uses `PORT` env if provided)

## Environment

- `VITE_API_BASE_URL` (optional): API server base URL (defaults to `http://localhost:3001`).
